# 📚 Library Management System

A simple **Library Management System** developed using **Python** and **MySQL**. This project allows users to manage books in a library by adding, viewing, issuing, and returning books.

---

## 🚀 Features

- ➕ Add New Book
- 📖 View All Books
- 📕 Issue Book
- 📗 Return Book
- 💾 MySQL Database Integration

---

## 🛠️ Technologies Used

- Python 3
- MySQL
- mysql-connector-python
- Git
- GitHub

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Vinithsripadhi/library-management-system.git
```

### 2. Open the project

```bash
cd library-management-system
```

### 3. Install MySQL Connector

```bash
pip install mysql-connector-python
```

### 4. Configure the database

Update the database credentials in `database.py`.

Example:

```python
host="localhost"
user="root"
password="your_password"
database="library"
```

### 5. Run the project

```bash
python main.py
```
