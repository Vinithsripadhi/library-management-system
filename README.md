# Library Management System

A simple Library Management System developed using Python and MySQL.

## Features

- Add Book
- View Books
- Issue Book
- Return Book

## Technologies Used

- Python
- MySQL

## Project Structure

library-management-system/
│── add_book.py
│── database.py
│── issue_book.py
│── return_book.py
│── view_books.py
│── main.py
│── library.py

## How to Run

1. Install Python.
2. Install MySQL.
3. Create the database.
4. Run:

```bash
python main.py
```
